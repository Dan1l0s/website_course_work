The macOS family of operating systems is the second most common for the desktop (after Windows). macOS market share (all versions are taken into account) as of February 2022, it is about $15.76% according to StatCounter estimates. The most popular version of macOS is Catalina (83.36% among all macOS versions), followed by Mojave (5.64%), High Sierra (5.05%), Sierra (2.1%), El Capitan (1.56%), Yosemite (1.21%).


Features:

The internal model of the Quartz graphics layer interacts well with the Portable Document Format (PDF), making it possible to output PDF to various devices.

Full-color zoomable icons.

Shadows around windows and isolated text elements give a sense of depth and visually separate the contents of one window from others.

The menus available for all applications are spell checking, a palette of special characters, color selection, font selection and a dictionary.

Anti-aliasing when displaying widgets, text, graphics, and window elements.

New interface elements, such as drawers and dialog modal windows for the document.

A new concept of the task switch — dock.

"Floating" focus (Interweaving windows) when switching between windows of various applications[15].

ColorSync, which is used for color matching, is built into drawing, printing and multimedia libraries.

OpenGL is used to display windows on the screen, which allows you to use hardware acceleration. This technology, which appeared in version 10.2 of Jaguar, is called Quartz Extreme.

Spaces — support for multiple desktops.

Dashboard, which appeared in version 10.4 of Tiger, supports small applications called widgets, which can be called and hidden with a single click.
